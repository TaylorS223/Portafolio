Cómo usar imágenes en HTML

Aprende a integrar imágenes en tus páginas web con la etiqueta <img>.

Código básico:

<img src="foto.jpg" alt="Descripción de la imagen" width="600" />

Ejemplo visual:

Atributos comunes:

  • src: ruta o URL del archivo de imagen.
  • alt: texto alternativo para accesibilidad y respaldo.
  • width / height: define el tamaño en píxeles o porcentaje.
  • title: muestra un texto emergente al pasar el cursor.
  • loading: con valor lazy para carga diferida (mejora el rendimiento).
